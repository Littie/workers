# Worker application

## Console command path

```
php bin/console
```
## Command list

### Check if worker has specific ability
*  can:designer ability
*  can:developer ability
*  can:manager  ability
*  can:tester ability

**Possible ability names:** 
- conversationWithManager
- draw
- setTasks
- testCode
- writeCode

### Get worker's abilities list
*  worker:designer
*  worker:developer
*  worker:manager
*  worker:tester
